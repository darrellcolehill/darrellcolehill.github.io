Brainrot Blocker is a **Chrome extension** that helps you stay focused by blocking short-form content on social media platforms like YouTube Shorts, Instagram Reels, and Facebook Watch. Stop the endless scrolling and regain your productivity! 





## Features

- Block short-form videos on popular platforms:
  - YouTube Shorts
  - Instagram Reels
  - Facebook Watch
- Customize which platforms to block
- Optionally replace blocked videos with a motivational message
- Lightweight and easy to install
